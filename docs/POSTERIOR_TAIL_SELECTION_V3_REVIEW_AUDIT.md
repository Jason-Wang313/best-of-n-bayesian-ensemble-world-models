# Posterior-Tail Selection V3 Review Audit

Target manuscript: `Posterior-Tail Selection in Bayesian Ensemble World Models`.

Purpose: harden the paper as a distinct Bayesian/ensemble world-model mechanism
study, not a duplicate wrapper around candidate-budget scaling. The manuscript
identity is posterior-tail aggregation: posterior samples, selected residuals,
ensemble dispersion, OOD mass, calibration residuals, stress sweeps, learned
bootstrap return ensembles, and lower-confidence selection.

## Side-by-Side Distinctness

- LLM paper: language response pools and score-rank evaluation.
- WAM paper: world-action rollouts and score-tail audits.
- Memory paper: stale retrieval provenance and memory-impostor rollouts.
- This paper: posterior-tail selection in Bayesian ensemble world-model rollouts.

The public title, abstract, contribution list, theorem role, diagnostics, and
repair center posterior aggregation. Candidate-budget maximization remains the
stressor, not the project identity.
